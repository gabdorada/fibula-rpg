# 📜 Fibula RPG

Projeto simples de RPG em Python feito para praticar lógica de programação, combate em turnos e uso de funções.

## 🚀 Como executar

```bash
git clone https://github.com/seu-usuario/fibula_rpg.git
cd fibula_rpg
python main.py
```

## 🎮 Funcionalidades

- Criação de personagem
- Sistema de combate com monstros
- Progressão de nível
- Uso de itens (poções)
- Sistema de experiência (EXP)

## 🧠 Conceitos praticados

- Estruturas condicionais (`if`, `else`)
- Laços de repetição (`while`)
- Funções
- Listas
- Módulo `random`
